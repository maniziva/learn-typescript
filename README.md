## ðŸš€ Getting Started

### 1. Clone the Repo

```bash
git clone https://github.com/maniziva/learn-typescript.git
cd learn-typescript
````

### 2. Install Dependencies
```bash
npm install
```

### 3. Run the TypeScript Compiler
```bash
npx tsc       # Compiles all .ts files in the src/ folder
```

### 4. Execute the Compiled JavaScript (optional)
```bash
node dist/basics.js
```
